# Learned personal styles

`scripts/learn_film_style.sh --install` writes reviewed personal scan styles here. Each generated JSON file is an ordinary data-driven stock definition and is discovered recursively by `src-tauri/build.rs`.

The generated dossier and evidence report describe these as interpretations of a combined positive-scan look, not measured characterizations of a film stock. See [`docs/learned-film-styles.md`](../../../docs/learned-film-styles.md) for the capture protocol, command-line workflow, and inference limits.
